# Open Face Chinese Poker

This project is a learning-focused implementation of the Open Face Chinese Poker
(Pineapple OFC variant).

Goals:
- Implement the basic rules and gameplay
- Allow a human player to play the game
- Later: add tools to analyze decisions and expected values (EV)

Constraints:
- Built step by step, prioritizing clarity over performance
- Intended for someone learning software development and GitHub

Rules reference:
https://www.pokernews.com/poker-rules/pineapple-open-face-chinese-poker.htm
